# Task 0.11 — Set Up Route Structure

## Overview

Create the complete route structure for the application using Next.js App Router route groups, including auth pages, admin dashboard, and user dashboard with placeholder pages.

**Estimated Time:** 3 hours
**Priority:** P1 (Blocker)
**Dependencies:** Task 0.9 completed (Layout components created)

---

## Prerequisites

- [ ] Task 0.9 completed (Layout components available)
- [ ] Task 0.8 completed (Auth helpers available)
- [ ] Task 0.10 completed (Design system ready)

---

## Route Structure Overview

```
app/
├── (auth)/                    # Auth routes (no sidebar)
│   ├── login/
│   ├── signup/
│   ├── forgot-password/
│   └── reset-password/
├── (admin)/                   # Admin routes (admin sidebar)
│   └── admin/
│       ├── layout.tsx
│       ├── page.tsx           # Admin dashboard
│       ├── members/
│       ├── courses/
│       ├── announcements/
│       └── settings/
├── (user)/                    # User routes (user sidebar)
│   ├── layout.tsx
│   ├── dashboard/
│   ├── courses/
│   ├── announcements/
│   └── profile/
├── api/
│   └── health/
├── layout.tsx                 # Root layout
├── page.tsx                   # Landing page (redirects)
├── not-found.tsx
└── error.tsx
```
